1. Install
   1. ```docker pull amazon/dynamodb-local```
   2. ```docker run -p 8000:8000 amazon/dynamodb-local```
   3. ```npm install -g dynamodb-admin```
   4. ```dynamodb-admin```
2. Run
   1. ``` AWS_REGION=dummy AWS_ACCESS_KEY_ID=dummy AWS_SECRET_ACCESS_KEY=dummy dynamodb-admin```
